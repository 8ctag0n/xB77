# Light Binary Bundle

This folder is the single source location for local Light Protocol validator binaries.

Required files:
- `light_system_program_pinocchio.so`
- `light_compressed_token.so`
- `account_compression.so`
- `light-localnet.env`
- `LIGHT_STACK_DUMP.md`

Used by:
- `scripts/localnet/start-validator-light.sh`

Canonical stack dump:
- `containers/light/bin/LIGHT_STACK_DUMP.md`
