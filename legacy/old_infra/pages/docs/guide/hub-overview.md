# The CFO Hub: Observer & Control Plane

The **Hub** is the central nervous system of the xB77 ecosystem for the human operator. It is a local-first dashboard that provides real-time observability and governance over your autonomous agents.

## 1. Commercial Terminal
This is the operational view. It allows you to:
- **Monitor Activity:** A live feed of agent transactions with "Privacy Toggles" to mask or reveal shielded data.
- **Forensic Radar:** A visual scanner that shows the risk assessment performed by the agent before a payment is executed.
- **Agent Thought Stream:** A real-time log of the agent's internal reasoning (e.g., "Analyzing optimal route," "Risk detected," "Shielding assets").

## 2. Shadow Governance
This panel is the security guardrail. When an agent triggers a **Lockdown Mode**, the transaction appears here. 
- **Inspect Encrypted Intents:** Decrypt the agent's intent using a local master key.
- **Manual Override:** Approve or Reject high-value transactions that exceed autonomous limits.

## 3. Control Plane
For developers and operators:
- **Agent Management:** Register new MCP agents, inspect their capabilities, and monitor their uptime.
- **Process Control:** Spawn or stop agent processes directly from the UI.
- **Tool Runner:** Manually trigger any MCP tool (e.g., `agent.pay`, `cfo.treasury.rebalance`) to test the system.

# El Hub del CFO: Plano de Observación y Control (ES)

El **Hub** es el panel central para el operador humano. Proporciona observabilidad y gobernanza en tiempo real.

## 1. Terminal Comercial
- **Monitoreo:** Feed en vivo con "Toggles de Privacidad".
- **Radar Forense:** Visualización del análisis de riesgo del agente.
- **Flujo de Pensamiento:** Registro de la lógica interna del agente.

## 2. Gobernanza en las Sombras
- **Inspección:** Desencripta las intenciones del agente.
- **Override:** Aprueba o rechaza transacciones bloqueadas por seguridad.

## 3. Plano de Control
- **Gestión:** Registra y monitorea agentes MCP.
- **Herramientas:** Ejecuta herramientas manualmente para pruebas.
