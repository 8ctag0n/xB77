Demo Unificada xB77

Este contenedor de nginx expone un único puerto 7777 y dirige cada ruta interna hacia los tres procesos principales:

  • /hub/ → bun hub/index.ts (dashboard).
  • /listener/ → bun run mcp/src/listener.ts (estado y webhooks).
  • /agent/ → bun run mcp/src/http.ts (MCP HTTP agent).
Merchant Terminal Listener /health Agent /health
Pasos de preparación:
  1. Inicia cada componente en tu host (puertos 7777, 7002, 7001).
  2. Construye y arranca este proxy: podman build -t xb77-demo-proxy containers/demo-proxy y podman run -p 7777:7777 xb77-demo-proxy.
  3. Abre http://localhost:7777 y empieza la demo.

Para la grabación puedes usar este proxy como punto único de conexión. Ajusta HUB_PORT, LISTENER_PORT y MCP_PORT si necesitas puertos personalizados.
