# Teaser Plan: xB77 'Cyber-Sovereign'

## Concepto
"The 24/7 Autonomous Sovereign." Un clip de 30 segundos, rápido, con cortes al ritmo de una música electrónica minimalista y elegante. Mostramos el agente moviéndose entre Solana, Sui y EVM como si fuera un operador de bolsa, dentro de una terminal cyberpunk.

## Storyboard
- **0-3s**: HOOK — "ALPHA: 5.35% APY" (neón púrpura). Texto: "Autonomía Financiera. Sin intermediarios."
- **3-10s**: MOTOR — Ráfaga de comandos (Zig Kernel, ZK-Proof verde parpadeante). Texto: "Zig-powered Core. ZK-Auditable Alpha."
- **10-20s**: ACCIÓN — Pantalla 3-split (Solana/Sui/EVM) con Ghost Receipts saltando. Texto: "Solana. Sui. EVM. Operaciones Atómicas."
- **20-27s**: SEGURIDAD — Smartphone con Sentinel Pulse + biométrica. Texto: "Soberanía total. Aprobación biométrica."
- **27-30s**: CTA — Logo xB77, QR code. Texto: "xB77. El futuro es autónomo."
