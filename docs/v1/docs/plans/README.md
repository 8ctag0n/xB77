## Status Definitions

- Stable: Code implemented, tests passing, ready for integration.
- Pending: Design complete, implementation in progress or not started.
- Blocked: Critical dependency failure (e.g., local validator, SDK bug) halting progress.
- Strategic: High-level architectural alignment.
