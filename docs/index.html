Sovereign Agent OS

xB77 is not a smart contract. It is a Terminal-Native Financial Operating System designed for the Agentic Economy. Written entirely in Zig and Rust, it allows autonomous AI agents to negotiate, settle, and audit themselves without relying on Web2 cloud infrastructure.

📱 Telegram Voice Agent (ElevenLabs)

When connected to a local node, you can interact with your agent via Telegram. Send it a text message, and the agent's local LLM will synthesize a voice response via ElevenLabs, reporting on its Swarm status.

Quick Start

# 1. Compile the Zig workspace
zig build

# 2. Run the Gateway Server
zig build run -- gateway &

# 3. Setup the Merchant Identity
./zig-out/bin/xb77 merchant setup-shop

Architecture

The system is composed of heavily optimized, low-level components:

  • Z-Node (C/Zig): Connects directly to Solana RPCs (Yellowstone gRPC) to stream blockchain events with sub-millisecond latency.
  • Sovereign Vault: AES-GCM encrypted local key management. No SaaS holds your keys.
  • MagicBlock HFT: Uses ephemeral rollups for instant transaction settlement.

Swarm Intelligence (P2P)

Agents in xB77 don't work alone. They communicate over a custom binary protocol (Agent Wire Protocol - AWP) via TCP sockets.

If an agent runs out of Sovereign Credits (Austerity Mode), it broadcasts a LoanRequestMsg to the Swarm. Other agents use their local LLM (Gemma 4) to evaluate the risk and automatically provide a Flash Loan via MagicBlock.

ZK Ghost Receipts

Corporate auditing without exposing business logic. The agent generates a Plonk ZK-Proof using Noir that mathematically proves 2.011% tax was paid on-chain.

The public commitment is stored on Solana. The auditor decrypts the real values (Amount, Recipient) locally in the browser using the Gateway's WASM bridge and a Viewing Key.
